## Italiano

### Descrizione
**GeoPhoto → KMZ** è uno strumento web gratuito che permette di estrarre le coordinate GPS incorporate nelle immagini e di esportarle come file **KMZ**, apribile direttamente con **Google Earth**.

Funziona interamente nel browser — nessuna installazione, nessun server, nessun dato inviato online.

### Funzionalità
- Estrazione automatica di latitudine, longitudine e altitudine (m s.l.m.) dai metadati EXIF/XMP
- Supporto per numerosi formati immagine: JPEG, TIFF, GeoTIFF, HEIC, HEIF, WEBP, PNG, DNG e tutti i principali formati RAW (CR2, CR3, NEF, NRW, ARW, SR2, RW2, ORF, RAF, PEF, X3F)
- Tabella interattiva con ordinamento e ricerca per nome file
- Esportazione in **KMZ** per Google Earth con opzioni personalizzabili (icona marker, altitudine 3D, anteprima immagine nel popup)
- Esportazione in **CSV** compatibile con Excel, QGIS e altri GIS
- Copia tabella negli appunti
- Funziona su PC, Mac, tablet e smartphone (Safari, Chrome, Firefox)

### Come si usa
1. Apri il link dell'app nel browser
2. Trascina le immagini nell'area di caricamento oppure clicca per selezionarle
3. Consulta la tabella con tutte le coordinate estratte
4. Clicca **Esporta CSV** oppure **Crea KMZ per Google Earth**

## English

### Description
**GeoPhoto → KMZ** is a free web tool that extracts GPS coordinates embedded in images and exports them as a **KMZ file**, ready to open in **Google Earth**.

It runs entirely in the browser — no installation, no server, no data sent online.

### Features
- Automatic extraction of latitude, longitude and altitude (m a.s.l.) from EXIF/XMP metadata
- Support for many image formats: JPEG, TIFF, GeoTIFF, HEIC, HEIF, WEBP, PNG, DNG and all major RAW formats (CR2, CR3, NEF, NRW, ARW, SR2, RW2, ORF, RAF, PEF, X3F)
- Interactive table with sorting and search by filename
- **KMZ** export for Google Earth with customizable options (marker icon, 3D altitude, image thumbnail in popup)
- **CSV** export compatible with Excel, QGIS and other GIS software
- Copy table to clipboard
- Works on PC, Mac, tablet and smartphone (Safari, Chrome, Firefox)

### How to use
1. Open the app link in your browser
2. Drag and drop images into the upload area or click to select them
3. Browse the table with all extracted coordinates
4. Click **Export CSV** or **Create KMZ for Google Earth**
